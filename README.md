# UnicodeVideoPlayer

## cara menggunakan
1. build App.csproj
2. di folder hasil build masukkan video ke dalam folder ./video
3. video yang disimpan di dalam folder ./video/hide tidak akan di putar
4. pastikan berekstensi .mp4 dan video memiliki suara
5. maksimal tinggi video adalah 186px
6. disarankan video tidak lebih dari 20 menit tergantung kapasitas ram

## fitur pemutar video
1. tekan panah kanan di keyboard untuk percepat 10 detik
2. tekan panah kiri di keyboard untuk mundur 10 detik
3. tekan space bar untuk pause dan resume video
4. tekan esc untuk langsung menghentikan video
5. video diputar secara urut ascending mulai dari 0123456789abcdefghijklmnopqrstuvwxyz
